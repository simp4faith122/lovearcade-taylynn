# Love Arcade — Taylynn

A small landing page scaffold for the Love Arcade — Taylynn project.

## Contents
- index.html — basic landing page
- styles.css — simple styling
- script.js — small client-side script
- LICENSE — MIT license
- .gitignore — common ignores for Node, macOS, and editors

## Usage
Open `index.html` in a browser to view the landing page. For development, serve the folder using your preferred static server (e.g. `npx serve` or `live-server`).
